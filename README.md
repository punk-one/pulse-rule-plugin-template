# pulse-rule-plugin-template

Community template for Pulse Rule Engine plugins.

## Ecosystem Repositories

- Engine: https://github.com/punk-one/pulse-rule-engine
- SDK: https://github.com/punk-one/pulse-rule-plugin-sdk
- Official plugins: https://github.com/punk-one/pulse-rule-plugins-official
- Plugin index: https://github.com/punk-one/pulse-rule-plugin-index
- Plugin template (this repo): https://github.com/punk-one/pulse-rule-plugin-template
- RFC governance: https://github.com/punk-one/pulse-rule-rfcs

## Compatibility Matrix

- Canonical matrix: https://github.com/punk-one/pulse-rule-plugin-index/blob/main/compatibility-matrix.md

## Baseline

- Template baseline version: `v1.0.0`.
- SDK dependency baseline: `github.com/punk-one/pulse-rule-plugin-sdk v1.0.0`.

## Breaking Changes

- Breaking changes are supported via major version bump only.
- Every major update must include RFC reference and migration notes.
- Compatibility matrix must be updated in `pulse-rule-plugin-index`.
